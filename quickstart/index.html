    Quickstart¶

    Let's take a brief whirlwind tour of some of the basic tools in exoatlas. Most of the concepts seen here are decribed in more detail later, but hopefully this is enough for you to get started!

    In [1]:
    Copied! 
    import exoatlas as ea
import exoatlas.visualizations as vi
import astropy.units as u 

ea.version()
    import exoatlas as ea import exoatlas.visualizations as vi import astropy.units as u ea.version()
    Out[1]:
    '0.6.1'

    🌌 Make Populations of Planets¶

    exoatlas can create objects containing properties for large samples of planets. Solar System planets come from JPL Solar System Dynamics, and exoplanets come from the NASA Exoplanet Archive.

    In [2]:
    Copied! 
    solar = ea.SolarSystem()
solar
    solar = ea.SolarSystem() solar
    Out[2]:
    ✨ Solar System | 8 elements ✨
    In [3]:
    Copied! 
    exoplanets = ea.Exoplanets() 
exoplanets
    exoplanets = ea.Exoplanets() exoplanets
    Out[3]:
    ✨ Exoplanets | 5862 elements ✨

    🧮 Extract Quantities and Uncertainties¶

    For a population, exoatlas can provide easy access to archival table columns and their uncertainties. All quantities have astropy.units attached to them, to minimize confusion about unit conversion.

    In [4]:
    Copied! 
    exoplanets.mass()
    exoplanets.mass()
    Out[4]:
    $[4914.8985,~4684.8142,~1131.1513,~\dots,~3257.7412,~162.09249,~616.90494] \; \mathrm{M_{\oplus}}$
    In [5]:
    Copied! 
    exoplanets.mass_uncertainty()
    exoplanets.mass_uncertainty()
    Out[5]:
    $[39.41072,~794.575,~37.503755,~\dots,~635.65681,~50.852545,~42.906835] \; \mathrm{M_{\oplus}}$

    exoatlas can also calculate derived quantities and use astropy.uncertainty to propagate uncertainties for those derived quantities.

    Copied!

    exoplanets.insolation()
    Out[6]:
    $[93243.949,~156505.06,~156786.2,~\dots,~721.81522,~61680.406,~203775.14] \; \mathrm{\frac{W}{m^{2}}}$

    ⚗️ Filter Populations by Properties¶

    exoatlas can extract subsets of populations, selecting planets just in a specific range of interest for particular properties. These subsets have all the same powers as their parent populations. By setting attributes for these subset populations, we can affect how they'll be visualized.

    In [9]:
    Copied! 
    nearby = exoplanets[exoplanets.distance() < 30*u.pc]
nearby.label = 'Nearby'
nearby.color = 'coral'
nearby.marker = 'P'
nearby.size = 20
nearby
    nearby = exoplanets[exoplanets.distance() < 30*u.pc] nearby.label = 'Nearby' nearby.color = 'coral' nearby.marker = 'P' nearby.size = 20 nearby
    Out[9]:
    ✨ Nearby | 487 elements ✨
    In [10]:
    Copied! 
    easy = exoplanets[exoplanets.transmission_snr(telescope='JWST', wavelength=4*u.micron) > 10]
easy.label = 'Easy'
easy.color='orchid'
easy.marker='*'
easy.outlined=True 
easy.filled=False 
easy.size=100
easy.annotate_planets = True
easy
    easy = exoplanets[exoplanets.transmission_snr(telescope='JWST', wavelength=4*u.micron) > 10] easy.label = 'Easy' easy.color='orchid' easy.marker='*' easy.outlined=True easy.filled=False easy.size=100 easy.annotate_planets = True easy
    Out[10]:
    ✨ Easy | 9 elements ✨

    🎨 Visualize Populations Together¶

    exoatlas provides some built-in visualizations, as well as a framework for constructing new plots that can easily compare multiple populations to each other.

    In [12]:
    Copied! 
    ps = vi.PlanetGallery()
ps.build([solar, exoplanets, nearby, easy, first])
    ps = vi.PlanetGallery() ps.build([solar, exoplanets, nearby, easy, first])

    Gosh, what a complicated plot! But at least it demonstrates lots of ways of visualizing populations!
